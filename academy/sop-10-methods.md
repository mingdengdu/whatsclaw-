## METHOD 06：WhatsApp 群发营销代理

**月收入目标**：¥2,000–8,000  
**起步成本**：¥0  
**难度**：⭐⭐

### 核心定位
帮商家做 WhatsApp 精准群发（节假日促销、新品通知、会员回访）。

### 起步步骤
1. **建立发送能力**：OpenClaw + WhatsApp，可群发给通讯录
2. **找第一个客户**：微商/电商/实体店
3. **设计报价单**（见下）
4. **执行**：导入客户名单 → 撰写文案 → OpenClaw 批量发送 → 汇报数据
5. **复购**：每月促销节点都需要，客户自然复购

### 报价单
| 服务 | 价格 |
|------|------|
| 文案撰写（1条） | ¥200 |
| 500条群发 | ¥300 |
| 2000条群发 | ¥800 |
| 月度营销套餐（4次发送） | ¥2,000/月 |

### 注意事项
- 只发同意接收消息的用户（避免封号）
- 发送间隔设置随机（3-8秒），模拟人工
- 不发敏感词（赌博/贷款/涉黄）

- 3个客户月套餐 → ¥6,000/月
