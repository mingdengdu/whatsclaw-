🧠 这3天真正学到的
LESSON 01
$1 真的够了。腾讯云 Lighthouse 香港节点，每月不到¥10，可以跑完整 Node.js + Nginx + AI Bot 全链路。
LESSON 02
不需要"准备好了再开始"。产品图用代码生成，课程内容边做边写，先上线再完善。
LESSON 03
收款链路是第一优先级。Domain → Landing Page → Stripe → 微信支付，这条链路跑通后，一切才有意义。
LESSON 04
AI 工具是放大器，不是替代品。你提供方向和判断，AI 提供执行力。3天完成的事，原来需要3周。
LESSON 05
安全 > 速度。.env 文件被 git 追踪是最大的坑，差点把 Stripe Live Key 推到 GitHub 公开仓库。
LESSON 06
内容即产品。10个案例页、客户故事页——这些内容本身就是最好的销售工具，比任何广告都有效。
