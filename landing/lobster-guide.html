1
安装 OpenClaw
3分钟完成本地安装，支持 Mac / Windows / Linux
1
安装 Node.js（v18+）
去 nodejs.org 下载 LTS 版本，或用 nvm 管理多版本。
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash nvm install 20 nvm use 20
2
全局安装 OpenClaw
一行命令，包含 Gateway 和 CLI 工具。
npm install -g openclaw
✅ 国内用户建议先 npm config set registry https://registry.npmmirror.com 切换淘宝镜像
3
初始化并启动
首次运行会引导你选择模型和配置 API Key。
openclaw init openclaw gateway start
4
打开 Web UI
默认在本地 3000 端口，浏览器访问即可使用。
open http://localhost:3000 # 或手动打开浏览器访问 http://localhost:3000

避坑指南
走过的弯路帮你省下来
⚠️ 安全风险：AI Skill 投毒攻击
市场上存在恶意 Skill，安装后会读取你的文件、API Key 甚至私钥。只从官方 KnHub 安装 Skill，不要用来路不明的 Skill 文件。
API Key 不要直接写进 config 文件，用环境变量注入
公网部署记得设置 Gateway 访问密码，不要裸奔
Memory 文件不要存放密码、私钥等敏感信息
新 Skill 安装前先 Google 一下作者，看有没有可信记录
不明来源的 Skill.md 文件不要直接执行，可能有 prompt injection
Cloudflare Tunnel 公开暴露前先做 IP 白名单或 Access 鉴权
定期备份 workspace 目录，特别是 memory/ 文件夹

常见问题
最高频的10个问题
Q
OpenClaw 和 Claude / ChatGPT 有什么区别？
Claude/ChatGPT 是纯对话界面，OpenClaw 是一个可部署的 AI 代理框架——它能调用工具、接入消息平台、执行文件操作、有长期记忆、可以自动化任务。你可以理解为「给 AI 装了手脚」。
Q
需要多强的技术能力？
基础部署只需要会用命令行，能执行 npm install 就够了。高级玩法（自写 Skill、MCP Server）需要 Python 或 TypeScript 基础。
Q
用什么模型？贵不贵？
支持 Claude、GPT-4o、DeepSeek、Kimi、MiniMax、GLM 等。DeepSeek 目前性价比最高，每百万 token 约 ¥1，日常使用一个月花不到 ¥50。
Q
企微接入是个人版还是企业版？
目前主要是企业微信服务商API接入，需要有企业主体。个人版企微机器人能力有限。如果没有企业主体，推荐先用 Telegram 或 QQ。
Q
开源免费的吗？
核心框架开源免费（MIT 协议）。部分高级功能和官方云服务收费，但本地自部署完全免费。
