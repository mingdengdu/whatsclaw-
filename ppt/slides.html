LESSONS LEARNED

踩过的坑，你不用再踩

⚠️

坑 1：.env 文件进了 git

Stripe 真实密钥被提交到 GitHub，需要 git filter-branch 清理历史。教训：永远先配好 .gitignore

⚠️

坑 2：chat-api.conf 端口冲突

nginx 无法启动是因为旧配置文件占用了 4000 端口。教训：每次改 nginx 后立即 nginx -t 测试

✅

最有效：一键部署脚本

把 git pull + 文件同步 + nginx + pm2 打包成一个脚本，每次更新一条命令搞定

✅

最省钱：HTML/CSS 生成图片

用 Puppeteer 截图替代 AI 图片生成 API，产品图和截图完全免费，效果一样好
